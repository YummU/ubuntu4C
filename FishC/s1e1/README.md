#C语言概述 用C语言编写Hello world

知识点总结：
#1 linux命令行命令
	1. vim进入编辑界面:
	 1.1 i或enter : 进入编辑模式
	 1.2 esc : 退出编辑模式
	 1.3 shift+: : 输入指令
	  1.3.1 q : 不编辑的情况下退出
	  1.3.2 q! : 编辑的情况下退出
	  1.3.3 ! : 强制退出
	  1.3.4 wq : 保存并退出
	2. sudo : 获取root权限
	3. gcc : linux编译工具 
	 3.1 gcc test.c -o test : 将test.c文件编译,并另存为test
	4. ./test : 运行test
#2 C语言语法
	1. #include <stdio.h> : 预处理头文件
	2. printf("") : C语言打印,f:format 格式化
